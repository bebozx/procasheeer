# POS Cashier (WPF + Offline)

نظام كاشير بسيط و سريع للمطاعم، يعمل أوفلاين بالكامل باستخدام SQLite.

## المميزات الأساسية:
- شاشة مبيعات بلمسة واحدة (Touch-friendly)
- أصناف + إضافات + أحجام
- طاولات / سفري / دليفري
- ضريبة + خدمة + رسوم توصيل
- طباعة فواتير حرارية USB
- عملاء + نقاط
- تقارير يومية وشهرية
- مصروفات + مشتريات + صافي ربح
